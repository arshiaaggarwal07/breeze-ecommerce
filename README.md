# Breeze — Sustainable E-Commerce Platform

## Overview  
Breeze is a full-stack sustainable e-commerce platform designed to promote eco-friendly products and conscious consumerism. The application enables users to browse, search, and purchase environmentally responsible products through a seamless and responsive interface.

## Features  
* Search and filter products
* Add to cart and manage items
* User authentication (Login / Signup)
* Order placement and checkout
* Responsive and interactive UI

## Tech Stack
### Frontend  
* HTML
* CSS
* React.js
### Backend
* Node.js
* Express.js
### Database
* MongoDB (Mongoose)

## How to Run  
### 1. Clone the repository:
```git clone https://github.com/arshiaaggarwal07/breeze-ecommerce.git``` <br> ```cd breeze-ecommerce```
### 2. Run Backend:
  ```cd backend``` <br>
  ```npm install``` <br>
  ```npm start```
### 3. Run Frontend:
  ```cd frontend``` <br>
  ```npm install``` <br>
  ```npm start```
